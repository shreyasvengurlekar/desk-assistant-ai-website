# **App Name**: Desk Assistant AI

## Core Features:

- AI-Powered File Organization: Uses a locally-processed AI tool to suggest file organization, learning user preferences over time.
- Safe Suggestions with Approval: Requires user approval before any file changes are made, ensuring full control.
- Automatic File Categorization: Automatically categorizes files based on content and type for easier management.
- Activity Log and Undo: Maintains a log of all actions with a one-click undo feature for easy reversal of changes.
- AI Chat for File Search: Uses AI to allow users to search for files using natural language.
- Duplicate File Detection: Identifies and suggests removal of duplicate files to free up storage space.
- Waitlist Management: Implements a waitlist form that captures user information and timestamps in Firestore for early access updates.

## Style Guidelines:

- Primary color: Deep blue (#2962FF) inspired by the logo, creating a professional and trustworthy feel.
- Background color: Light gray (#F0F4F8), almost white, offering a clean, modern backdrop and strong contrast.
- Accent color: Sky blue (#74B9FF), drawing from the logo, used for subtle highlights and interactive elements.
- Font pairing: 'Inter' (sans-serif) for body text and headings, creating a clean, modern reading experience.
- Modern, minimalist icons with a blue accent to match the app's logo, enhancing usability.
- Clean, minimal layout with ample whitespace to provide clarity and focus on the core features, implemented in multi-page design
- Smooth fade-in sections, gentle button transitions, and hover micro-interactions to add a layer of polish and engagement.
