# Desk Assistant AI - Website

Welcome to the official website repository for **Desk Assistant AI**, the privacy-first, local-only file organization assistant for Windows.

🌐 **Live Website:** [https://deskassistantai.vercel.app/](https://deskassistantai.vercel.app/)

---

## 🚀 About the Project

Desk Assistant AI is designed to conquer file chaos on your Windows PC while keeping your data 100% private. This website serves as the central hub for users to learn about the project and download the latest version of the application.

### Key Features of the Application:
- 🔒 **Privacy First:** All file scanning and AI analysis happen 100% locally on your machine. No cloud uploads.
- 💻 **Local Working:** Works perfectly offline. Your files never leave your computer.
- 🛡️ **Safe & Transparent:** Permission-first design. Every action requires your approval and is completely undoable.
- 💬 **AI Chat Search:** Find files using natural language (e.g., "Find the invoice from last Tuesday").
- 📥 **Tray Mode:** Runs quietly in the system tray with minimal resource usage.

---

## 🛠️ Technical Implementation

The website is a modern Next.js application designed for performance and dynamic updates.

### Tech Stack:
- **Framework:** [Next.js 15 (App Router)](https://nextjs.org/)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/) & [Shadcn UI](https://ui.shadcn.com/)
- **Dynamic Content:** Integrated with the [GitHub API](https://docs.github.com/en/rest) to automatically fetch the latest release versions, file sizes, and changelogs.
- **Rendering:** [React Markdown](https://github.com/remarkjs/react-markdown) for parsing and styling GitHub release notes.
- **Backend:** Firebase (Firestore) for managing contact form submissions and the waitlist.
